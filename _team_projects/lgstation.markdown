  - 이가현(LG전자 DX School): 발표자, BX, CX 전체 내용 작성
- 프론트엔드 1명
  - 이승현(LG전자 DX School): 데이터 크롤링, 프론트엔드 개발, 화면 설계
- 백엔드 & 클라이언트 1명
  - 김현태(LG전자 DX School): 백엔드 개발, 클라이언트 개발, 데이터베이스 설계, 데이터 분석
