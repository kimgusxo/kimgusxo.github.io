	- Java 11
	- Spring Boot 2.7.8
	- Gradle
	- Spring Data JPA
	- MySQL
	- Firebase Authentocation
	- Firebase Cloud Message

### Model Server
	- Python 3.8
	- Flask
	- OpenCV
	- EasyOCR
	- Sklearn
	- Pandas
